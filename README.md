# Habit Tracking Application

## Overview
This is a command-line Habit Tracking Application built in Python.
It allows users to create habits, track completions, and analyze streaks.

## Features
- Create daily and weekly habits
- Mark habits as completed
- Calculate longest streak
- Filter habits by periodicity
- View longest streak across all habits
- SQLite database persistence
- Unit tests included

## Project Structure
- habit.py → Habit model
- repository.py → Database logic
- analytics.py → Functional analytics
- cli.py → User interface
- tests/ → Unit tests
- screenshots/ → Proof of functionality

## How to Run
python main.py

## Run Tests
python -m unittest discover tests

## Database
SQLite database stored locally as habits.db
